Wpfsharp.Globalizer

Better globalization, both localization and styles, for WPF projects

View the Project on GitHub rhyous/WPFSharp.Globalizer

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

A library for runtime localization of WPF. This project decouples localization from build as well as providing the technology to change the language at runtime, including the ability to apply a separate style for any given language.

Use XAML Resource files that can be easily updated with any text editor. Never include localization in your code freeze again.
